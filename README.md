# Flashly Box 🗃️

**Flashly Box** คือแอปพลิเคชันแฟลชการ์ดเพื่อการศึกษาสมัยใหม่ระดับพรีเมียมที่ออกแบบมาเพื่อการเรียนรู้ที่มีประสิทธิภาพ โดดเด่นด้วยธีมสี Earth-tone ที่สวยงาม แอนิเมชันที่ลื่นไหล และระบบหลังบ้านที่ทรงพลังโดย Supabase

## ✨ คุณสมบัติ

- **การจัดการชุดข้อมูล (Collection Management)**: จัดระเบียบเนื้อหาการเรียนรู้ของคุณออกเป็นหมวดหมู่ที่ชัดเจน
- **แฟลชการ์ดแบบโต้ตอบ**: เรียนรู้อย่างมีประสิทธิภาพด้วยอินเทอร์เฟซแฟลชการ์ดที่ใช้งานง่ายและมีแอนิเมชัน
- **การออกแบบระดับพรีเมียม**: สร้างขึ้นด้วยจานสี Earth-tone ที่คัดสรรมาอย่างดีและไมโครแอนิเมชันที่ลื่นไหล
- **การซิงค์ข้อมูลแบบเรียลไทม์**: ระบบยืนยันตัวตนที่ปลอดภัยและการเก็บรักษาข้อมูลผ่าน Supabase
- **เทคโนโลยีที่ทันสมัย**: ใช้ฟีเจอร์ล่าสุดของ Vue 3 และ Tailwind CSS 4

## 🛠️ เทคโนโลยีที่ใช้

- **Framework**: [Vue 3 (Beta/Vapor)](https://vuejs.org/)
- **Bundler**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Motion for Vue](https://motion.dev/vue)
- **State Management**: [Pinia](https://pinia.vuejs.org/)
- **Backend/Auth**: [Supabase](https://supabase.com/)
- **Data Import**: [XLSX](https://sheetjs.com/)

## 🚀 เริ่มต้นใช้งาน

### สิ่งที่ต้องมีก่อน

- Node.js (v20.19.0 หรือ >=22.12.0)
- npm

### การติดตั้ง

```sh
npm install
```

### การพัฒนา

รันเซิร์ฟเวอร์สำหรับการพัฒนาพร้อมการอัปเดตแบบเรียลไทม์:

```sh
npm run dev
```

### การสร้างสำหรับใช้งานจริง (Production Build)

คอมไพล์และบีบอัดไฟล์สำหรับใช้งานจริง:

```sh
npm run build
```

### การทดสอบ

รัน Unit Test ด้วย [Vitest](https://vitest.dev/):

```sh
npm run test:unit
```
